---
description: Encuentra contradicciones entre notas del vault y las resuelve o marca como pregunta abierta
---

Tema opcional a enfocar: $ARGUMENTS (si no se da, escaneá todo el vault).

1. Buscá en `conocimiento/`, `proyectos/` y `context/objetivos.md` pares de afirmaciones que se contradigan (algo dicho en una nota vieja que una más nueva desmiente o cambia).
2. Para cada contradicción, evaluá:
   - ¿Cuál fuente es más nueva?
   - ¿Es una contradicción real o una evolución de pensamiento (cambiar de opinión no es contradicción)?
3. Resolvé cada una:
   - **Ganador claro:** actualizá la nota desactualizada, agregando una línea breve tipo "Antes decía X (ver nota vieja), actualizado a Y por Z".
   - **Ambiguo:** creá una nota corta en `conocimiento/` tipo `Conflicto - <tema>.md` documentando ambas posiciones, marcada `#pendiente`, para que el usuario decida.
4. Reportá: contradicciones auto-resueltas (con el cambio), y las marcadas como pendientes para decisión humana.

El vault no debería tener dos notas que se contradigan sin saberlo.
